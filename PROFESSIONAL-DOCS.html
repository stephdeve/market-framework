üöÄ Installation

Method 1: Automatic Installation (Recommended)

1
Download the Framework

Clone or download the framework to your development machine

Terminal
cd "d:\Mes projets"
git clone https://github.com/your-repo/market-framework framework
2
Create a New Project

Use the project creator script to set up a new application

Terminal
php framework\create-project.php my-app
cd my-app
3
Configure Database

Edit config/database.php with your MySQL credentials

config/database.php
return [
    'host' => 'localhost',
    'dbname' => 'my_database',
    'username' => 'root',
    'password' => 'your_password'
];
4
Create Database

Create your MySQL database

MySQL
mysql -u root -p
CREATE DATABASE my_database CHARACTER SET utf8mb4 COLLATE utf8mb4_unicode_ci;
EXIT;
5
Start Development Server

Launch the PHP built-in server

Terminal
composer serve
# Or manually:
php -S localhost:8000 -t public
6
Visit Your Application

Open your browser and navigate to:

‚úì Success! Visit http://localhost:8000

Optional: Install Authentication

Terminal
# Copy the auth installer
copy "..\framework\install-auth.php" .

# Run the installer
php install-auth.php

# Run migrations
php migrate.php

# Authentication is ready!
# Visit: http://localhost:8000/register

Optional: Install Tailwind CSS

Terminal
# Copy the Tailwind installer
copy "..\framework\install-tailwind.php" .

# Run the installer
php install-tailwind.php

# Install dependencies
npm install

# Start Tailwind watch mode
npm run dev
